# Order Blocks Strategy

Order block — зона последней противоположной свечи перед импульсным движением.

Bullish OB:
- последняя bearish свеча перед сильным ростом;
- цена возвращается в зону;
- появляется реакция/объём/структура.

Bearish OB:
- последняя bullish свеча перед сильным падением;
- цена возвращается в зону;
- появляется отказ от зоны.

Подтверждения:
1. BOS/CHOCH рядом с OB.
2. Снятие ликвидности перед реакцией.
3. Volume spike или сильная реакционная свеча.
4. FVG рядом усиливает сетап.

No trade:
- зона слишком широкая;
- нет реакции;
- цена уже много раз тестировала OB;
- invalidation слишком далеко.
