# Default Trading Risk Rules

1. Не финансовая рекомендация.
2. Не формировать сигнал без stop/invalidation.
3. Минимальный ориентир R/R: 1:2.
4. Не смотреть пары Bybit futures с 24h volume < 100000 USDT.
5. Если есть конфликт сигналов — watch-only.
6. Confidence high разрешён только при совпадении нескольких факторов: structure + volume + level + invalidation.
7. Если данных мало — no trade.
